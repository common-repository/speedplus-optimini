== Description ==

Simple and useful plugin for optimizing website loading speed. All settings are collected on one page.


= Main features =

* Unload scripts and styles: Emojis, Dashicons.
* Disable in Header: RSS, Windows Live Writer, RPC RSD, api.w.org, shortlink, WP version.
* Disable HTML in comments or Disable all comments totally.
* Turn off oEmbed, Remove links to previous and next posts.
* Minify HTML code.
* Sequrity: Disable login hints, Disable loading admin bar in frontend, Create correct robots.txt, Disable app passwords.
* WooCommerce: Optimize intervals of Action Scheduler's log cleaning and Facebook feed generation.
* WooCommerce: Disable cart fragments on front page, Disable WooCommerce marketing garbage, Hide out of stock Woocommerce products in shop/categories/tags/related.
* Disable automatic plugins and themes updates.
* SEO: Remove cyclic links.
* Make jQuery passive.
* Languages: English, Ukrainian, Russian.

== Installation ==
1. Upload plugin folder to the /wp-content/plugins/ directory. OR download plugin in .zip format and install it in WordPress.
1. Activate the plugin through the ‘Plugins’ menu in WordPress.
1. Go to "Settings => SpeedPlus Optimini" and use checkboxes to customize your optimization.

== Frequently Asked Questions ==
= Is it safe to use this plugin? =

Any site optimization means making changes to its work, correcting the working code. It's always a risk.
Moreover, each site is unique, and therefore it is impossible to foresee all the options and exclude all errors.
This plugin does only what is stated in the description. Read the description of the optimization checkboxes carefully before activating them, make sure you understand what you are doing and everything will be fine.
In any case, you use this plugin at your own risk.
